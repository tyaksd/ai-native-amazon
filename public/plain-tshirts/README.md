# Plain T-Shirt Images

This directory contains pre-made plain t-shirt images for each color variant.

## Required Images

Place the following plain t-shirt images in this directory:

- `black-plain.png` - Black (#0e0e0e) plain t-shirt
- `white-plain.png` - White (#ffffff) plain t-shirt  
- `navy-plain.png` - Navy (#0f1830) plain t-shirt
- `grey-plain.png` - Grey (#d1d2d6) plain t-shirt
- `dark-heather-plain.png` - Dark Heather (#424848) plain t-shirt
- `red-plain.png` - Red (#FF1B2B) plain t-shirt
- `blue-plain.png` - Blue (#2665CC) plain t-shirt
- `sand-plain.png` - Sand (#d8c5a9) plain t-shirt
- `natural-plain.png` - Natural (#fff6ea) plain t-shirt
- `military-green-plain.png` - Military Green (#686f54) plain t-shirt

## Image Specifications

- Size: 1024x1024 pixels
- Format: PNG
- Style: High-quality e-commerce product image
- Content: Plain t-shirt with no design, graphics, or text
- Background: Pure white (#FFFFFF)
- Position: Centered, front-facing, laid flat

## Usage

These images are used as base templates for compositing generated designs. The AI will overlay the design PNG onto these plain t-shirt images to create the final product photos.
